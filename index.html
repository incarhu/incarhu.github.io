安卓环境配置

ANDROID_HOME:`C:\android-sdk`

JAVA_HOME:`C:\Program Files (x86)\Java\jdk1.8.0_101`

CLASSPATH:`.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;`

Path:`;%ANDROID_HOME%\tools;%ANDROID_HOME%\platform-tools;`

移动混合APP打包

为项目添加一个平台：ionic platform add android

在模拟机上运行：ionic emulate android

打包成安卓：ionic build android

GitHub上传

先提交项目：git commit -m "first commit"

添加到远程分支：git remote add origin https://github.com/incarhu/aaa.git

推上去：git push -u origin master
