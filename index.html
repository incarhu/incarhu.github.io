移动混合APP打包

为项目添加一个平台：ionic platform add android

在模拟机上运行：ionic emulate android

打包成安卓：ionic build android

推上去：git push -u origin master

MYSQL

1.连接数据库：cd到当前目录，，mysql -h localhost -u 用户名 -p 密码

2.显示数据库：show databases

3.创建一个数据库：create database test

4.使用库：use test

4.创建表：create table msg

6.查看库下面的表：show tables

7.查看表结构：desc msg;

Mongodb

1、运行数据库,开机：--首先在一个cmd里输入 mongod --dbpath c:\data\db ，然后另开一个cmd键入 mongo

2、列出所有数据库：--show dbs

3、使用某个数据库：--use 数据库名字，如果新建数据库，也是use。use一个不存在的，就是创建

4、插入数据：--db.student.insert({"name":"xiaoming","age":"20","sex":"nan"});

5、查看数据集合：--db.student.find();

6、查看数据库的所有集合：--show collections

7、导入外部集合json：mongoimport --db demo --collection student --drop --file C:\Users\1.json

  --db demo (导入到哪个数据库)

  --collection student (导入到哪个集合)

  --drop (删除已存在的数据)

  --file C:\Users\1.json (需要导入哪个文件的路径)

8、查找数据，用find：----db.student.find({"score.math":70},);
